# abc
## Лабораторная работа №1
Организация работы стека
C++: Нужно вручную управлять памятью (использовать new/delete или std::vector). Код подробный из-за указателей и шаблонов. Быстрый, но легко ошибиться с памятью.

Java: Память чистится автоматически (сборщик мусора). Использует исключения для ошибок, строгие типы данных. Простой и безопасный код, часто с ArrayDeque для скорости.

Python: Всё просто — использует встроенный list или deque. Код короткий, без забот о памяти. Динамические типы, но работает медленнее для больших стеков.
