## Лабораторная работа №1
Организация работы стека
В C++ стек управляется вручную через выделение памяти на стеке (локальные переменные) и куче (new/delete), что требует контроля от разработчика. В Java все объекты создаются в куче (new), а стек хранит только примитивы и ссылки, управление памятью осуществляется сборщиком мусора. В Python стек используется для хранения ссылок на объекты, при этом все объекты размещаются в динамической памяти, управляемой сборщиком мусора, а работа со стеком скрыта от программиста.
