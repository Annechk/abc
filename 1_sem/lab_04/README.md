## 1. Деревья

### Общие принципы работы с деревьями

**Дерево** — иерархическая структура данных, где каждый узел имеет:
- Один родительский узел (кроме корневого)
- Ноль или более дочерних узлов
- Ключ (значение) для упорядочивания

**Основные операции:**
- Вставка элемента
- Поиск элемента
- Обход дерева (DFS, BFS)
- Поиск пути от корня к узлу

### Python
**Особенности реализации:**
- Динамическая типизация
- Использование ссылок (None для отсутствующих узлов)
- Рекурсивные алгоритмы для обходов
- Простота реализации благодаря сборке мусора

```python
class TreeNode:
    def __init__(self, value):
        self.val = value
        self.left = None
        self.right = None
```

**Алгоритмы:**
- Вставка: итеративный спуск с поиском места
- Поиск пути: рекурсивный DFS с backtracking
- Обходы: рекурсивные DFS и итеративный BFS

### C++
**Особенности реализации:**
- Статическая типизация с указателями
- Явное управление памятью (new/delete)
- Использование nullptr вместо NULL
- Шаблонные классы для обобщенности

```cpp
class TreeNode {
public:
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int value) : val(value), left(nullptr), right(nullptr) {}
};
```

**Алгоритмы:**
- Вставка: итеративный спуск с двойными указателями
- Деструктор для рекурсивного удаления
- Использование STL контейнеров для путей

### Java
**Особенности реализации:**
- Сборка мусора (автоматическое управление памятью)
- Использование null для отсутствующих узлов
- Коллекции Framework для работы с путями
- Дженерики для типобезопасности

```java
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    public TreeNode(int value) {
        this.val = value;
        this.left = null;
        this.right = null;
    }
}
```

**Алгоритмы:**
- Вставка: итеративный спуск с parent tracking
- Использование ArrayList для путей
- Queue интерфейс для BFS

### Сравнительный анализ операций с деревьями

| Операция | Python | C++ | Java |
|----------|---------|------|-------|
| Вставка | O(h) итеративная | O(h) с указателями | O(h) итеративная |
| Поиск | O(h) рекурсивный | O(h) итеративный | O(h) итеративный |
| DFS обход | O(n) рекурсивный | O(n) рекурсивный | O(n) рекурсивный |
| BFS обход | O(n) с deque | O(n) с queue | O(n) с LinkedList |
| Поиск пути | O(n) с backtracking | O(n) с vector | O(n) с ArrayList |

## 2. Графы

### Общие принципы работы с графами

**Граф** — структура данных, состоящая из:
- Вершин (узлов)
- Рёбер (связей между вершинами)
- Весов рёбер (для взвешенных графов)

**Основные алгоритмы:**
- Алгоритм Дейкстры (кратчайшие пути)
- Обход в глубину (DFS)
- Обход в ширину (BFS)

### Python
**Особенности реализации:**
- Списки смежности как вложенные списки
- Использование heapq для приоритетной очереди
- Float('inf') для бесконечности
- Словари для гибкости

```python
self.adjacency_list = [[] for _ in range(vertices)]
```

**Алгоритм Дейкстры:**
- Приоритетная очередь на heapq
- Множество посещенных вершин
- Обновление расстояний при найденном коротком пути

### C++
**Особенности реализации:**
- Векторы для списков смежности
- STL priority_queue с greater comparator
- numeric_limits<int>::max() для бесконечности
- Явное управление памятью

```cpp
vector<vector<Edge>> adjacencyList;
```

**Алгоритм Дейкстры:**
- Использование pair<int, int> для очереди
- Векторы для расстояний и посещений
- Восстановление пути через previous массив

### Java
**Особенности реализации:**
- ArrayList<ArrayList<Edge>> для списков смежности
- PriorityQueue с Comparator
- Integer.MAX_VALUE для бесконечности
- Автоматическое управление памятью

```java
private List<List<Edge>> adjacencyList;
```

**Алгоритм Дейкстры:**
- Кастомный класс Pair с Comparable
- Массивы для расстояний и visited
- Восстановление пути через previous массив

### Сравнительный анализ алгоритма Дейкстры

| Характеристика | Python | C++ | Java |
|----------------|---------|------|-------|
| Сложность | O((V+E)logV) | O((V+E)logV) | O((V+E)logV) |
| Структура данных | heapq | priority_queue | PriorityQueue |
| Представление графа | список кортежей | вектор структур | ArrayList классов |
| Бесконечность | float('inf') | numeric_limits | Integer.MAX_VALUE |
